# logo

1. Download [`toys_r_us.ttf`](https://www.cdnfonts.com/toys-r-us-2.font) from CDNFonts:
    ```
    curl -O https://fonts.cdnfonts.com/s/95273/toys_r_us.ttf
    ```
2. Upload it to [google-font-to-svg-path](https://danmarshall.github.io/google-font-to-svg-path/) and download the SVG file with the following settings:
    * Text: WikindeR
    * Size: 100
    * Separate Characters: Yes
    * Stroke Width: 0
    * Non-Scaling Stroke: No
4. Edit the SVG file in a text editor as per [Encycolorpedia](https://encycolorpedia.com/companies/us/toys-r-us).
